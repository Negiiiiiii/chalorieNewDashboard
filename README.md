# 食事管理アプリ - ダッシュボード

## 概要
このディレクトリには、日々の栄養摂取を管理・可視化するための単一ページの食事管理アプリケーションが含まれています。HTMLファイル1つで完結するスタンドアロン型のWebアプリケーションです。

## ファイル構成
- `nutrition_dashboard.html` - メインアプリケーションファイル（HTML/CSS/JavaScript）

## 主な機能

### 1. 食事スコア表示
- 円形プログレスバーで1日の食事スコアを視覚的に表示（0-100点）
- 日付と共にスコアを表示
- グラデーション効果を使った美しいUI

### 2. キャラクター「ペンネ」
- オリジナルキャラクター「ペンネ」（鳥型）が皿の上でアニメーション
- まばたきアニメーション（3秒間隔）
- ふわふわ浮遊アニメーション
- 吹き出しでアドバイスメッセージを表示

### 3. 週間カレンダー
- 7日間の週間カレンダービュー
- 各日の達成状況を色分け表示：
  - 緑：目標達成
  - 赤：目標未達成
  - グレー：未来の日付
- クリックで日付を切り替え可能
- アクティブな日付は白い背景で強調表示

### 4. 栄養素トラッキング
5つの主要栄養素を視覚的に管理：
- カロリー（kcal）
- 炭水化物（g）
- 脂質（g）
- 食物繊維（g）
- 塩分（g）

各栄養素は以下の要素で表示：
- アイコン（絵文字）
- 現在値/目標値
- プログレスバー（アニメーション付き）
- グラデーション背景で色分け

### 5. UIコントロール
- 設定ボタン（⚙️）
- 編集ボタン（✏️）
- ページネーションドット（複数ページ対応の準備）

## デザイン特徴

### カラースキーム
- 背景：ピンク〜オレンジのグラデーション
- UI要素：白ベース + グラスモーフィズム効果
- アクセント：パステルカラー

### アニメーション
- スコアの円形プログレスバー（1秒）
- 栄養バーのプログレス（順次表示）
- キャラクターの浮遊・まばたき
- ボタンのホバー・アクティブエフェクト

### レスポンシブデザイン
- モバイルファーストデザイン
- 最大幅500pxのコンテナ
- 15pxのページパディング

## 技術スタック
- **HTML5** - セマンティックマークアップ
- **CSS3** -
  - Flexboxレイアウト
  - グラデーション・影効果
  - アニメーション（@keyframes）
  - グラスモーフィズム（backdrop-filter）
- **JavaScript (Vanilla)** -
  - DOMイベントハンドリング
  - プログレスバーアニメーション
  - インタラクティブ要素の制御

## インタラクティブ要素

### イベントリスナー
1. カレンダー日付クリック → 日付切り替え
2. ページネーションドットクリック → ページ切り替え
3. 設定・編集ボタンクリック → コンソールログ出力
4. 自動まばたき → 3秒間隔でアニメーション

### 動的データ
現在はハードコードされた値を使用：
- スコア：75点
- カロリー：1500/2000kcal
- 炭水化物：2500/1000g（超過）
- 脂質：1500/2000g
- 食物繊維：100/2000g
- 塩分：2300/2000g（超過）

## 使用方法
1. `nutrition_dashboard.html` をWebブラウザで開く
2. カレンダーで日付を選択して、その日の栄養データを表示
3. ペンネからのアドバイスメッセージを確認
4. 各栄養素の摂取状況をプログレスバーで確認
